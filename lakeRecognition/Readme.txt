Python code to find landing point in lake depending of weather

geoFit.ods -> resolution of the map depending of the latitude
getSatImage.py -> HTTP GET from google static map API
getWeather.py -> get weather from yahoo weather API and open weather API, output is a python dictionnary for both API
lakeClass.py -> class definition of a lake
lakeDetection.py -> main python script
mapClass.py -> class definition of a map
windConversion.ods -> wind degrees conversion to fit classic cartesian angle
